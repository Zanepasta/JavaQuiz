Aufgabe programmieren Sie ein Quiz in Java.
(Ausgabe auf der Konsole ist ausreichend)

1. Das Programm soll den User zuerst ein Hauptmenü anzeigen das Ihn fragt ob er spielen möchte oder nicht.
	Wählt der User nicht spielen aus soll das Prgramm beendet werden. Der User soll über beendigung informiert werden.
2. Erstellen sie 10 Fragen die dem User der Reihe nach angezeigt werden mit dazugehörigen antwort möglichkeiten.
	Der user soll eine Antwort auswählen und angezeigt bekommen ob die Antwort richtig war oder nicht.
	Wenn alle 10 Fragen abgearbeitet wurden soll der User zurück ins Hauptmenü gelangen.
3. Die Fragen sollen Randomisiert werden (+Antwortmöglichkeiten) => zufälliges Aufrufen der Fragen und zufälliges anzeigen der antwortmöglichkeiten
4. Implementieren sie eine Eingabeüberprüfung.
