Aufgabe programmieren Sie ein Quiz in Java.
(Ausgabe auf der Konsole ist ausreichend)

1. Das Programm soll den User zuerst ein Hauptmenü anzeigen das Ihn fragt ob er spielen möchte oder nicht.
	Wählt der User nicht spielen aus soll das Prgramm beendet werden. Der User soll über beendigung informiert werden.
2. Erstellen sie 10 Fragen die dem User der Reihe nach angezeigt werden mit dazugehörigen antwort möglichkeiten.
	Der user soll eine Antwort auswählen und angezeigt bekommen ob die Antwort richtig war oder nicht.
	Wenn alle 10 Fragen abgearbeitet wurden soll der User zurück ins Hauptmenü gelangen.
3. Die Fragen sollen Randomisiert werden (+Antwortmöglichkeiten) => zufälliges Aufrufen der Fragen und zufälliges anzeigen der antwortmöglichkeiten
4. Implementieren sie eine Eingabeüberprüfung. Das Programm soll nicht fortfahren wenn eine Fehlerhafte Eingabe getätigt wurde 
	(die Eingabe einer Falschen Antwort ist keine Fehlerhafte Eingabe)
5. Implementieren sie Punkte in das Quiz. Entscheiden Sie selbständig über die Art der Punkte vergabe.
6. Erstellen sie ein temporäres Leaderboard (nach Neustart des Programms ist das Leaderboard wieder im Default zustand)
7. Implementieren sie einen Temporären Highscore (bei neustart setzt er sich auf 0)
8. Implementieren sie eine Bestenliste. Wählen sie selbstständig wieviele Einträge der Bestenliste angezeigt werden. 
	Geben sie dem User die Möglichkeit sich in die Bestenliste einzutragen.
9. Erweitern sie Ihr Menü so dass der User ein neues Spiel starten kann, sich den Highscore anzeigen lassen kann und die Bestenliste. 
	Entscheiden Sie selbst wie die Darstellung und Auswahl der entsprechenden Punkte erfolgt.
10. Ändern sie den Highscore und die Bestenliste so das sie Sitzungsübergreifend erhalten bleibt.
11. Erweitern Sie Ihre Fragenliste um weitere 10+ Fragen (Es sollen weiterhin nur 10 Fragen pro Spiel abgefragt werden.)
12. Erweitern Sie Ihr Spiel um einen weiteren Spielmodus in dem der User eigenhändig die Antworten eingeben soll.
13. Erweitern Sie Ihr Spiel um eine Joker-Funktion die der Spieler zu einem beliebigen Zeitpunkt auslösen kann (in beiden Spielmodi). 
	Dieser Joker gibt dem Spieler die Richtige Antwort.
14. Implementieren sie eine zeitliche Begrenzung zur Fragen Beantwortung. Legen sie die Zeitspanne selbst fest.
15. Implementieren Sie eine zeitliche Begrenzung für einen gesamten Spieldurchlauf. 
	Die Zeitspanne soll so gewählt werden das wenn der User nie eine Frage beantwortet er nicht zum Ende des Spiels gelangt 
	(wählen sie selbständig eine Zeitspanne die diese Anforderung erfüllt)
